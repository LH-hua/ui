### 属性

| 参数 | 说明 | 类型 | 可选值 | 默认值 | 备注 |
| ---- | ---- | ---- | ---- | ---- | ---- |
| distributedConfigure | 流线图配置 | Object | —— | —— | —— |
| chartPieces | 自定义范围标注解释颜色 | Array | —— | —— | 代码有使用案例 |
| mapOptions | 地图配置 | Object | —— | —— | —— |
| backgroundOptions | 底图配置 | Array | —— | —— | —— |

### distributedConfigure 属性

| 参数 | 说明 | 类型 | 可选值 | 默认值 | 备注 |
| ---- | ---- | ---- | ---- | ---- | ---- |
| data | 绘制流线图的数据 | Array | —— | —— | 二维数组，一个数组代表一条线，数组中的第一个对象是起点和值，第二个对象是终点 |
| mapJson | 地点名称和经纬度的映射 | Object | —— | —— | —— |
| option | echarts图表配置 | Object | —— | —— | —— |

### distributedConfigure.option 属性
| 参数 | 说明 | 类型 | 可选值 | 默认值 | 备注 |
| ---- | ---- | ---- | ---- | ---- | ---- |
| pieces | 自定义范围标注颜色 | Object | —— | —— | —— |
| legendLabelColor | 视觉映射组件的文本颜色 | String | —— | —— | '#555' |
| series | 对应echarts的series配置项 | Array<object> | —— | —— | series[0]:渲染线条，类型为`lines`;series[1]:渲染点位，类型为`effectScatter`;可对应类型在echarts配置项中查找属性配置 |
| events | 为echart添加鼠标事件 | Array<object> | —— | —— | —— |


### option.events 属性
| 参数 | 说明 | 类型 | 可选值 | 默认值 | 备注 |
| ---- | ---- | ---- | ---- | ---- | ---- |
| type | 事件类型 | String | —— | —— | —— |
| target | 为echart某个series类型添加事件 | String | —— | —— | 传series的type名，如'series.line'，非必传 |
| callBack | 事件回调函数 | Function | —— | —— | 会拿到当前点击的echart数据 |
