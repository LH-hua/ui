# 介绍

| 部署方式 | 相关技术点 |
| ---- | ---- |
| 质控仪、动管仪、AI智能终端 | 帧差法、图像形态学处理（通道转换、滤波、二值化、位置权重）|

记录移动事件的起止时间，并自动截取移动变化量最大的关键帧进行判别。
适用于站房动态管控中对动物进入、人员进入等移动轨迹进行监测。移动侦测算法是判定异常入侵监测的算法基础。
移动侦测自动标记监控区域中进入室内的人员或入侵动物，并在后台记录移动关键帧、关键帧时间、移动持续时间、移动权值等移动侦测项目。
