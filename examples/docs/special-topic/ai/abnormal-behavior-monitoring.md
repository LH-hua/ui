# 介绍

| 部署方式 | 相关技术点 |
| ---- | ---- |
| 质控仪、动管仪、AI智能终端 | 移动侦测 、人形识别、人体姿态识别、百度API |

对站房、车间、厂区监控区域内的人员抽烟、看手机、采样造假、私接线路、改装设备、私自校准等怠工行为、异常违规行为进行实时监测识别和现场记录留证，为远程监管提供有力证据，落实责任。
标识异常行为的类型，后台记录行为关键帧、关键帧时间、摄像头、持续时间、置信度，异常行为识别的历史记录可在部署的硬件前端设备或平台上查询。
